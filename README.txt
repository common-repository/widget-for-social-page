== Description ==

It will embed your facebook page into a widget area. You can add a widget in any of your widget areas and It will show your facebook page.

== Installation ==

= From the dashboard: =
1. Go to the WordPress Dashboard "Add New Plugin" section
2. Search For "Widget for Social Page"
3. Install, then Activate it

= From the control panel: =
1. Unzip (if it is zipped) and Upload `widget-for-social-page` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Is there need anything to do with my theme? =

No, you can not do anything, It will be work automatically with your widget area.

= Does it work with any theme? =

Absolutely! It will work with any theme where WordPress works.
